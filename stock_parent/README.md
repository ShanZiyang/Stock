# stock_parent

#### 介绍
开发一个基于长短期记忆(LSTM)神经网络的股票预测平台，通过结合历史价格数据，提供更准确的股票价格预测。

#### 软件架构
该项目基于B/S框架，采用SpringBoot+Vue+Echarts进行可视化展示。基于LSTM神经网络进行股票价格的预测
